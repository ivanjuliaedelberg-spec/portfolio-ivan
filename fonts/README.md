Place the font files here. The CSS is already configured and waiting.

## Download

1. Go to: https://www.onlinewebfonts.com/fonts/Facundo
2. Download the variants you need (Regular, Light, Bold)
3. Convert/extract the `.woff` and `.woff2` files
4. Name them exactly as below and place them in this folder

## Expected filenames

```
FSPDEMOFacundo-Light.woff2
FSPDEMOFacundo-Light.woff
FSPDEMOFacundo-Regular.woff2
FSPDEMOFacundo-Regular.woff
FSPDEMOFacundo-Bold.woff2
FSPDEMOFacundo-Bold.woff
```

## Activate

Once the files are here, open `css/style.css` and uncomment the
`@font-face` block at the top of the file.
